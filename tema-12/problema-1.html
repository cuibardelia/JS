Problema 1

Declarati o variabila si initializati-o cu un array gol.

  1. Folosind functia .push, adaugati pe rand in array elementele:
    'red', 'blue', 'yellow', 'green', 'cyan'.
  2. Afisati in consola valoarea array-ului sub forma unui tabel (vezi console.table()).
  3. Sortati crescator elementele din array si afisati pozitia elementului cu valoarea 'green' din noul array.
  4. Afisati penultimul element din array.
  5. Stergeti ultimul element din array.
  6. Afisati stringul format din concatenarea tuturor elementelor din array separate prin '#'.
